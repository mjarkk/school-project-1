# school project 1
This is a repo for a school project

## Programs you will need:
- Only windows: [git](https://git-scm.com/)
- [node.js](https://nodejs.org/en/)
- [yarn](https://yarnpkg.com/en/)
- Google Chrome

## Install:
1. Clone the project
2. Open a terminal inside the project folder and type:
3. yarn install
4. yarn start
